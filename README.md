# 浏览器新标签页书签导航插件

Chrome MV3 · Edge · 新标签页替换 · 书签管理 · 导航工具

替换浏览器新标签页的多功能书签导航插件，支持分类管理、拖拽排序、Bing 每日背景、本地图片轮播、待办事项、值班提醒、多语言（中/英），兼容 Chrome / Edge / 360 浏览器。

地址：https://lms.im/magic/chrome-new-tab-bookmark-plugin/

## 功能

### 书签分类管理
- 按顺序渲染分类列表（左侧边栏）与书签卡片（右侧主区域）
- 点击分类标题栏展开/折叠（状态持久化）
- 新建、编辑、删除分类
- 每个分类显示链接数量计数
- 分类图标 Emoji 选择器

### 书签链接操作
- **添加链接**（两种方式）：
  - 弹窗手动填写：分类名、链接名、URL、图标、描述
  - 点击插件图标自动抓取当前页面标题、URL、favicon，只需选择分类
- **编辑、删除**已有链接
- **拖拽排序**：
  - 编辑模式下拖拽分类重新排序
  - 编辑模式下拖拽链接在分类内或跨分类移动
- 点击计数，按最常访问显示 Top Sites（最多 8 个）

### 存储模式
- **同步模式**（默认）：直接读取 Chrome 书签树，增删改同步到书签管理器，跨设备自动同步
- **本地模式**：数据存储于 `chrome.storage.local`，无配额限制
- 切换模式自动迁移书签、设置、分类元数据、点击计数

### 待办事项（Todo）
- 新增、编辑、删除待办事项
- 双击文字进入编辑
- 三级优先级：红（高）、橙（中）、绿（低），带对应颜色左边框和半透明背景
- 打勾标记完成，自动记录完成时间
- 清除已完成
- 导出 CSV（Level / Content / CreatedTime / DoneTime），可直接用 Excel 打开

### 值班提醒
- 配置 API 地址和 Key，自动获取排班数据
- 当天值班 → Toast 显示「今日值班」粗体 + 明日值班交接名单
- 明天值班 → Toast 提示「明日值班」
- API Key 自动附加到请求 URL

### 背景设置
- 纯色背景 + 透明度（RGBA 实时预览）
- Bing 每日一图（直接调用 Bing API）
- 自定义图片 URL
- 本地图片上传（IndexedDB 存储，支持多张轮播 30s 切换）

### 天气
- 显示当地实时温度、天气图标、城市名
- 优先浏览器定位 → 高德 IP 定位兜底
- 30 分钟缓存避免重复请求
- 数据来源：高德开放平台天气 API

### 搜索
- 实时模糊搜索书签名称和 URL
- `Ctrl+K` 或 `/` 快捷键聚焦
- 结果列表键盘导航（↑↓方向键 + Enter 打开）

### 每日诗词
- 从 jinrishici.com 获取每日诗词
- 显示诗句、朝代、作者

### 时钟
- 实时显示当前日期、时间、星期
- 自动更新，秒级刷新

### 数据导入导出
- **导出**：完整 JSON（含书签、设置、待办事项、分类元数据、点击计数），文件名带日期
- **导入**：恢复 JSON 备份（自动识别同步/本地模式写入书签数据 + 待办事项）
- **浏览器书签导入**（本地模式）：从 Chrome 书签管理器导入，可选择归入已有分类或按原文件夹导入

### 多语言
- 简体中文（默认）、英文
- 所有 UI 文本通过 `chrome.i18n.getMessage()` 国际化
- 支持 `textContent`、`placeholder`、`aria-label`、`title` 的翻译

### 无障碍
- 语义化 ARIA 标签和 role
- 键盘导航支持
- 屏幕阅读器友好

## 安装

1. 下载项目到本地并解压
2. 打开浏览器扩展管理页面（`chrome://extensions/` 或 `edge://extensions/`）
3. 打开开发者模式
4. 点击「加载已解压的扩展程序」，选择项目文件夹
5. 点击插件「详情」→「固定到工具栏」
6. 浏览器设置 →「启动时」→「打开新标签页」

## 文件结构

```
├── manifest.json          # MV3 清单（权限、CSP、默认语言）
├── newtab.html            # 新标签页主界面
├── newtab.js              # 新标签页交互逻辑（书签、待办、天气、搜索、设置）
├── popup.html             # 弹窗快速添加界面
├── popup.js               # 弹窗交互逻辑 + 分类自动补全
├── shared.js              # 共享工具（i18n / Modal / 书签树解析 / 存储管理）
├── background.js          # Service Worker（书签采集 + 书签变更通知）
├── styles.css             # 全部样式
├── _locales/
│   ├── zh_CN/messages.json  # 简体中文（~170 key）
│   └── en/messages.json     # 英文（~170 key）
└── icons/                   # 扩展图标（16/48/128）
```

## 兼容性

- Google Chrome（MV3）
- Microsoft Edge
- 360 浏览器

## 注意事项

- 需要 `storage`、`bookmarks`、`tabs`、`scripting`、`favicon` 权限
- 同步模式下书签数据直接由 Chrome 书签管理器管理，删除插件前请备份书签
- 清空浏览器数据可能导致书签丢失，请及时导出备份
- 天气和值班功能需要自行申请 API Key（高德开放平台 / 值班系统）
